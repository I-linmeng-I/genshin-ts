# 列表与字典

## list
- `list(t, [])` 明确空列表类型。
- 需要类型占位且保持引脚不连线时用 `list(t, 0)` / `list(t, null)`。
- 仅在可推断类型时使用 `list(0)` / `list(null)`。
- 列表方法（`map`/`filter`/`find` 等）有编译支持。

## dict
- `dict(...)` 生成只读字典。
- 需要类型占位且保持引脚不连线时用 `dict(k, v, 0)`。
- 仅在可推断类型时使用 `dict(0)` / `dict(null)`。
- 字典字面量最多 50 对键值。
- 可写字典请声明节点图变量并通过 `f.get` / `f.set`。
